## ğŸš€ Getting Started

### Prerequisites
- Node.js (version 18 or higher)
- npm (version 9 or higher) or yarn
- Expo CLI (installed globally)

### Installation
```bash
git clone https://github.com/your-username/edugram.git
cd edugram
npm install

```

### Deployment
```bash
# Web build
npm run build:web

# Mobile builds
npx expo build:android
npx expo build:ios
```
